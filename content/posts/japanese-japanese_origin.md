# 前言

很多人在看動漫或學習日文時，都會發現一個有趣的現象：日文有些發音聽起來很像英文的變體。為什麼漢字與西洋發音會同時存在於同一個語言中？在正式開始學習日文之前，就讓我們一起來聊聊日語的起源與它的故事吧!

# 日語的真正起源：日本語系

在語言學的分類上，日語屬於日本語系（Japonic languages），這是一個獨立的語系。
而在漢字傳入日本之前，日本本土原本就有自己的語言，這部分被稱為「和語（Wago）」。
和語的發音和中文完全沒關係，例如：

- 水（Mizu）
- 山（Yama）
- 吃（Taberu）

這時候可以說是一個 **有音無字** 的時期，擁有自己的語言（和語），但人們只能透過口頭交流，沒有任何符號可以將語言記錄下來。

那就衍生出了一個問題，**現在的平假名和片假名從何而來?**
為了解決會說不會寫的窘境，日本經歷了一段長達數百年的文字演化期。

# 漢字的傳入與假名的誕生

## 1. 漢字初傳的痛苦適應期（5～6世紀）

早在隋唐之前，漢字就已經傳入日本。剛接觸漢字時，日本人經歷了一段痛苦的 **純粹借用期**。當時他們不僅借用漢字的字義，還硬著頭皮使用中文的文法（主詞＋動詞＋受詞）來寫公文，但這跟日語母語的文法完全相反（主詞＋受詞＋動詞），日本人當然使用得苦不堪言。

為了解決這個問題，他們開始妥協，發展出了**變體漢文**。他們依然保留漢字的意思，但開始改用日文的文法順序來排列漢字（例如把「我吃飯」寫成「我飯食」）。雖然稍微好懂了一些，但這終究還是用「字義」在寫字，無法精準記錄日語特有的助詞與語氣。

## 2. 尋找專屬音標：大化革新與萬葉假名的誕生（西元 645 年起）

**西元 645 年**，日本推動**大化革新**。雖然名義上是一場政治與制度的改革，但它也間接加速了是假名的誕生。

當時日本派了大量**遣唐使**到中國學習，帶回了豐富的新詞彙（政治、建築、宗教、醫學等），這大量擴充了日語中的「漢語」詞彙，也將漢字的讀音深深烙印在日語的發音系統裡。

隨著中央集權建立，政府開始全面用漢字統治國家，但很快遇到了一個無法用**變體漢文**解決的瓶頸：**國家的歷史與神話怎麼寫？**

大化革新後，為了強化天皇統治的正當性，政府下令編纂國家歷史。但在記錄日本本土的神明名字、天皇族譜、各地地名以及古代流傳下來的和歌時，用純粹的漢字根本寫不出來，也無法表達原汁原味的日語情感。

為了解決這個痛點，日本人終於決定放棄漢字的**意思**，單純只取它的**發音**來當作拼音工具，這就促成了**萬葉假名**的誕生。

但漢字筆畫太多了，全部用來當注音符號實在太浪費時間了。 於是，為了求快、求方便，萬葉假名開始朝兩個不同的方向簡化，這就誕生了平假名與片假名。

## 平假名：把漢字「草書化」

平假名是從漢字的草書演變而來的。

當時的宮廷女性在寫和歌或私人信件時，為了書寫流暢、優美，會把萬葉假名的漢字用草書連筆快速寫下，久而久之，這些草書漢字就簡化成了固定的符號。 因為最初多為女性使用，且線條圓潤，所以平假名早期也被稱為**女手**。

| 現代平假名  | 原始漢字（萬葉假名） | 演變邏輯                                   |
| :---------: | :------------------: | :----------------------------------------- |
| **あ (a)**  |        **安**        | 「安」的草書，保留了寶蓋頭和女字的圓潤線條 |
| **か (ka)** |        **加**        | 「加」的草書，左邊的力與右邊的口連筆寫成   |
| **の (no)** |        **乃**        | 「乃」的草書，一筆畫繞圈到底               |
| **た (ta)** |        **太**        | 「太」的草書，保留了十字與下方的一點       |

## 片假名：把漢字「拆解」

片假名則是透過擷取漢字的偏旁或特定部分演變而來的（「片」在日文裡有不完整、片段的意思）。

當時的僧侶在聽師父講經，或者學生在閱讀漢文經典時，需要在字裡行間快速做筆記、標註發音。 因為書寫空間小且需要極高的速度，他們就直接拆解漢字，只寫漢字的某一個部位來代表那個音。 這也是為什麼片假名看起來比較方正、剛硬。

| 現代片假名  | 原始漢字（萬葉假名） | 演變邏輯（拆解部位）       |
| :---------: | :------------------: | :------------------------- |
| **ア (a)**  |        **阿**        | 取「阿」左邊的部首（阝）   |
| **イ (i)**  |        **伊**        | 取「伊」左邊的部首（亻）   |
| **ウ (u)**  |        **宇**        | 取「宇」上方的寶蓋頭（宀） |
| **エ (e)**  |        **江**        | 取「江」右邊的部件（工）   |
| **カ (ka)** |        **加**        | 取「加」左邊的部件（力）   |

這就是現在假名的由來。

# 大航海時代的驚奇（16 世紀中葉）

**主要影響語言：葡萄牙語、西班牙語**

西元 1543 年，一艘載著葡萄牙人的船漂流到了日本種子島，這是日本第一次接觸到歐洲人。 隨後，葡萄牙和西班牙的商人和傳教士大量來到日本，進行所謂的**南蠻貿易**。
他們帶來了當時日本人從未見過的生活用品、食物和基督教文化。 這些詞彙因為傳入時間太早，很多已經完全融入日本人的生活，甚至連日本人自己都常常忘記它們是外來語！

- **代表詞彙：**
  - **パン (Pan)**： 麵包（源自葡萄牙語 _pão_）
  - **ボタン (Botan)**： 鈕扣（源自葡萄牙語 _botão_）
  - **タバコ (Tabako)**： 香菸（源自葡萄牙語 _tabaco_）
  - **カステラ (Kasutera)**： 長崎蛋糕（源自葡萄牙語 _Castella_）
  - **カルタ (Karuta)**： 紙牌（源自葡萄牙語 _carta_）

# 鎖國時期的唯一窗口（17 世紀 ～ 19 世紀中葉）

**主要影響語言：荷蘭語**

進入江戶時代後，日本實行了嚴格的**鎖國政策**，把外國人都趕了出去。 但是為了掌握世界局勢與吸收先進知識，幕府特許荷蘭作為唯一可以通商的歐洲國家（地點限制在長崎的出島）。
這段時期，日本學者拼命透過荷蘭文獻學習西方的醫學、天文、物理和化學，這門學問被稱為「蘭學」。 因此，這時期的外來語多半與科學、醫學和航海有關。

- **代表詞彙：**
  - **コーヒー (Kōhī)**： 咖啡（源自荷蘭語 _koffie_）
  - **ガラス (Garasu)**： 玻璃（源自荷蘭語 _glas_）
  - **メス (Mesu)**： 手術刀（源自荷蘭語 _mes_）
  - **ランドセル (Randoseru)**： 小學生書包（源自荷蘭語 _ransel_，原本是軍用背包）

# 全面西化的明治維新（19 世紀下半葉 ～ 二戰前）

**主要影響語言：英語、德語、法語**

1853 年美國黑船來航，日本被迫開國，隨後展開了**明治維新**。 日本人發現自己在軍事和科技上大幅落後，於是開始瘋狂向各個西方強國學習。
這個時期非常有趣，日本會根據不同國家的強項來引進專門的詞彙：

- **向德國學醫學與哲學：**
  - **カルテ (Karute)**： 病歷表（源自德語 _Karte_）
  - **ギプス (Gipusu)**： 石膏（源自德語 _Gips_）
  - **アレルギー (Arerugī)**： 過敏（源自德語 _Allergie_）

- **向法國學藝術與時尚：**
  - **アンケート (Ankēto)**： 問卷調查（源自法語 _enquête_）
  - **デッサン (Dessan)**： 素描（源自法語 _dessin_）

- **向英美學商業與科技：** 英語詞彙開始大量進入，例如: **ドア (Doa，門)**、**ナイフ (Naifu，刀子)**。

# 美軍進駐與全球化大爆發（1945 年二戰後 ～ 至今）

**主要影響語言：美式英語**

第二次世界大戰後，日本被盟軍（主要是美軍）佔領，美國的流行文化、電影、音樂、生活方式如海嘯般湧入日本。 加上後來的 IT 革命和全球化，現代日語的新外來語，有 90% 以上都來自英文。
現在的日本人遇到新的科技名詞或外國概念，已經很少像明治時代那樣辛苦地去翻譯成漢字了，而是直接用片假名拼出英文發音：

- **代表詞彙：**
  - **コンピューター (Konpyūtā)**： Computer（電腦）
  - **スマートフォン (Sumātofon) / 簡稱 スマホ (Sumaho)**： Smartphone（智慧型手機）
  - **インターネット (Intānetto)**： Internet（網路）

- **有趣的變體——「和製英語」**：
  日本人甚至厲害到用英文單字自己發明了英文裡根本沒有的詞彙！ 例如：
  - **サラリーマン (Sararīman)**： Salary + Man = 上班族
  - **スキンシップ (Sukinshippu)**： Skin + ship = 肌膚之親、身體接觸

# 總結

總結來說，日語在字體與發音上受到了多國語言深遠的影響。 從單純借用漢字，過渡到萬葉假名（純粹將漢字作為音標），接著誕生了本土的假名，最後再大量融入西方外來語。 這是一套經過漫長歷史洗禮與淬鍊，同時交錯著漢字、平假名與片假名，既獨特又充滿魅力的語言系統！
